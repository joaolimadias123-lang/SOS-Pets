# Sistema de Gestão para ONG de Proteção Animal

## Descrição da ONG

Este projeto foi desenvolvido para auxiliar uma ONG de proteção animal no gerenciamento de doações, adotantes e animais disponíveis para adoção.

## Objetivo

Centralizar informações da ONG em um sistema simples e eficiente.

## Tecnologias Utilizadas

* Node.js
* MySQL
* HTML
* CSS
* JavaScript
* GitHub

## Instalação

1. Clone o repositório.
2. Configure o arquivo .env.
3. Instale as dependências.
4. Execute o sistema.

## Estrutura do Projeto

/docs → Documentação

/backend → Backend

/frontend → Interface do usuário

/database → Scripts do banco de dados
